# Cost-Aware-Prediction
Final Term Project for CDA 5106 Advanced Computer Architecture at the University of Central Florida (Spring 2026)

## Contents
This github contains the code for running our GRU and LSTM. When evaluating our work, please ignore everything in the "Not Being Used" Folder, as none of that pertains to our LSTM or GRU. That folder contains scripts for our old tabnet, mlp, and lstm models. These algorithms are not a part of our final product and are not required to be run to get our results. You only need to run our GRU and LSTM to get results. The LSTM in "Not Being Used" is not the LSTM we used for our results.

## Running the Models
To run the models you first need to create a python environment, then install the requirements via our requirements.txt file. To install the requirements run the following command:

pip install -r requirements.txt

## Our Data
The raw data that we used for results can be found in lstm.log and gru.log in the LSTM and GRU folders respectively.
